javscript -> object.js
静态成员
keys(对象),得到对象的属性名的数组
values(对象),得到都对象的所有属性值的数组
entries(对象),得到对象所有属性名和属性值的数组
实例成员
toString:得到某个对象的字符串格式
toString可以被重写
valueOf方法:得到某个对象的值 默认情况下返回当前对象本身
在JS中当自动进行类型转换时,如果对一个对象进行类型转换时实际上先调用对象的valueof 方法然后在调用toString方法
function
所有函数都有function中的实例成员
arguments 是类数组也称为伪数组,伪数组缺少实例方法
arguments数组中的值会对形参进行对应
获取函数形参数量 functionName.length

function 中的apply
JavaScript -> call_apply
apply 改变this 的指向，call改变this指向
apply 入参 传递形参的时候 数组
call 入参 传递形参的时候 直接传参
通常可以利用apply和call方法去将某个伪数组转为真数组
